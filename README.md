# Enterprise AI Support Assistant

An AI-powered Production Support Assistant built using Spring AI and Ollama.

## Features

- AI-powered Incident Analysis
- Knowledge Base Search
- Recommendation Engine
- Hybrid AI Orchestration
- RAG with ChromaDB
- Conversation Memory
- SQL Server Integration
- Spring JDBC
- REST APIs
- GitHub Integration

## Technology Stack

- Java 21
- Spring Boot 3.5.x
- Spring AI 1.1.7
- Ollama (Qwen2.5:7B)
- ChromaDB
- SQL Server 2025 Express
- Spring JDBC
- Maven
